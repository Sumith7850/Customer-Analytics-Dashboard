# Customer Analytics Dashboard

##  Overview

This project focuses on analyzing customer data to generate actionable business insights using a combination of SQL, Python, and Power BI.

The goal is to transform raw data into meaningful visualizations that support better decision-making and business strategy.

---

##  Tech Stack

* **SQL** → Data extraction and querying
* **Python (VS Code)** → Data cleaning, transformation, and analysis
* **Power BI** → Interactive dashboard and visualization

---

## Project Workflow

1. **Data Extraction**

   * SQL queries are used to retrieve relevant customer data.

2. **Data Processing & Analysis**

   * Python (Pandas, NumPy) is used for:

     * Data cleaning
     * Handling missing values
     * Exploratory Data Analysis (EDA)

3. **Data Visualization**

   * Processed data is visualized using Power BI dashboards.
   * Provides insights into customer behavior and key metrics.

##  How to Run the Project

1. Execute SQL queries from:

   ```
   sql/customer_queries.sql
   ```
2. Run the Jupyter Notebook:

   ```
   notebooks/customer_analysis.ipynb
   ```
3. Open the Power BI dashboard:

   ```
   dashboard/customer_dashboard.pbix
   ```
